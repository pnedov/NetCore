ASP.NET MVC Core 2.0 / Release date: 29 Oct 2019
ASP MVC NET Core project MoviesBox.ADO with integrated REST API functionality
The web application make search by films title calling the Omdbapi.com API, storing selected movie into MSSQL DB and display it by request.
(DB layer project working on ADO.NET technologie)
  1. Source code is stored here C# source code
  2. Open the project using solution file 'MoviesBox.ADO.sln'
  3. Open MoviesBox/appsettings.json file and place your conn. string into Key 'ConnectionStrings'
  4. Execute SQL scripts for creating DB ant tables. Use sql scripts from here db_and_tables.sql
  5. Run web application in Visual Studio IIS Express
  6. Task N4 : C# Task 4.docx
